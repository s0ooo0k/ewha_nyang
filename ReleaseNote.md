# 이화냥탐험대🐱

이화냥탐험대는 이화여자대학교를 배경으로 온,오프라인을 넘나드는 AR 퍼즐 게임입니다. 
<br>
게임에 대한 더 많은 이야기는 [REAMDE](README.md)를 참고해주세요!
<br>
<br>


## Features
- "정문" 스테이지의 뷰포리아를 활용한 AR 게임
- "학문관", "ECC" 스테이지의 AR 게임
- "대강당" 스테이지의 Face Tracking AR 미션
- "포스코관", "본관", "도서관" 스테이지의 2D 게임
- 각 스테이지 클리어 시 고양이 뱃지 수집 가능
